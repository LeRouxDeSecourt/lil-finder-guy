# lil finder guy

Hey! I made lil finder guy in Blender :) I'm still learning shading and rigging, so feel free to download the model and make something cool with it!

## What is this
- A Blender model of a tiny character called "lil finder guy".
- Includes a base T-pose and a sample render.

## Files
- `little-finder-guy.blend` - the Blender file.
- `T-pose-little-finder-guy.png` - T-pose preview.
- `in-situation-little-finder-guy.png` - example render.

## How to use
1. Open `little-finder-guy.blend` in Blender.
2. Tweak materials, lighting, and rigging as you like.
3. Render, animate, or kitbash it into your own scenes.
